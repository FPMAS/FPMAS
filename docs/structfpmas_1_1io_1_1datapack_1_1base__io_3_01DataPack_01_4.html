Detailed Description

api::communication::DataPack base_io specialization.

The purpose of this specialization is to read / write a DataPack from / to a larger DataPack, efficiently copying raw binary data.

Writes source to the destination DataPack buffer at the given offset. pack_size() bytes are written, and offset is incremented accordingly.

First, source.size is written (datapack::pack_size<std::size_t> bytes) and then the source.buffer() char array is written contiguously.
