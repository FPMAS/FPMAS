Detailed Description

template<class T, typename LayerType = DefaultLayer, int N = 1>
class FPMAS::graph::base::Graph< T, LayerType, N >

Layered Graph class.

A layered Graph is a graph where each Node can simultaneously belong to several layers, so that each layer can be explored independently from each node.

Layered are represented as predefined arc types defined in the LayerType enum parameter. Each node then manages lists of incoming and outgoing arcs for each type.

Because layer types are provided at compile time as template parameters, managing the several layers can be done in constant time whatever the numbe of layers is.

In order to be valid, the LayerType and N template parameters must follow some constraints :

  • LayerType must be an enum using integers as underlying type, assigning positive values to each layer, starting from 0. The associated index can be thought as the level of each layer. Here is a valid example :
    enum ExampleLayer {
    DEFAULT = 0,
    COMMUNICATION = 1,
    ROAD_GRAPH = 2
    }
  • N must be equal to the number of layers defined by LayerType (in the example, N = 3)
