Detailed Description

template<typename AgentType>
struct fpmas::io::json::light_serializer< fpmas::api::utils::PtrWrapper< AgentType >, typename std::enable_if< std::is_base_of< fpmas::api::model::Agent, AgentType >::value &&std::is_default_constructible< AgentType >::value &&std::is_same< AgentType, typename AgentType::ConcreteType >::value >::type >

A default light_serializer specialization for default constructible Agent types.

When a default constructor is available for AgentType, this specialization is automatically selected.

The usage of this specialization is recommended, since its optimal: no json data is required to serialize / unserialize a light_json representing such an AgentType.
