Detailed Description

template<typename T, typename JsonType>
class fpmas::graph::JsonTemporaryNode< T, JsonType >

nlohmann::json based TemporaryNode implementation.

The json representing the node is stored within the class. If build() is called, a new node is allocated from the deserialized json.

Template Parameters
Tnode data type
JsonTypejson type, that defines deserialization rules (e.g.: nlohmann::json or fpmas::io::json::light_json)
