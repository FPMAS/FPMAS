Detailed Description

Runtime API.

The purpose of a Runtime is to execute jobs scheduled in a Scheduler.

Runs jobs from start, included, to end, excluded.

The Runtime is currently base on a unit time step. In consequence, the Runtime will execute jobs at dates start, start+1 and start+n for any integer n such that start+n < end.
