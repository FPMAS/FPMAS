template<typename T , typename IdType , int N>
void FPMAS::graph::base::Arc< T, IdType, N >::unlink

Unlinks this arc from its source and target nodes.

The arc is removed from the incoming / outgoing arcs list of its target / source nodes respectively. However, the source and target nodes pointer of this arcs stay valid. In consequence, this function is usefull to unlink / delete arcs at the Graph scale.
