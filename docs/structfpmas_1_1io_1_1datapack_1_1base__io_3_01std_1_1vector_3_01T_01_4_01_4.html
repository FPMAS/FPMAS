static

Writes vec to the data_pack buffer at the given offset. pack_size(vec) bytes are written, and offset is incremented accordingly.

First, vec.size is written (datapack::pack_size<std::size_t> bytes) and then each item in vec are written contiguously using the datapack::write() specialization for T.
