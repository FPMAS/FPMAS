85 /*
86 * Splits the first node of the list in two nodes, that become the
87 * children of the node.
88 *
89 * The zone is split vertically if node->height > node->width,
90 * horizontally otherwise, in order to minimize the size of
91 * frontiers.
92 *
93 * By construction, the leafs are ordered so that largest zones are
94 * at the beginning of the list, so its consistent to split those
95 * nodes first. Indeed, split nodes are always pushed back in the
96 * list, while the first element of leafs is popped. Moreover, at
97 * the beginning of the algorithm, the only element in `leafs` is
98 * `root`, that represents the whole grid.
