Detailed Description

template<typename T, int N>
class FPMAS::graph::base::Layer< T, N >

Class representing a Layer of a Graph.

Layers instances are used by each node to represent their incoming and outgoing arcs according to layers defined in the specified LayerType enum.

Template Parameters
Tassociated data type
Nnumber of layers
