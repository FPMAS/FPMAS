Detailed Description

template<class T>
class FPMAS::graph::parallel::synchro::None< T >

Synchronisation mode that can be used as a template argument for a DistributedGraph, where no synchronization or overlapping zone is used.

When a node is exported, its connections with nodes that are not exported on the same process are lost.
