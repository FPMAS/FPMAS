◆ distribute()

template<class T , template< typename > class S>
void FPMAS::graph::DistributedGraph< T, S >::distribute

Distributes the graph accross the available cores using Zoltan.

The graph is distributed using Zoltan::LB_Partition and Zoltan::Migrate. Query functions used are defined in the FPMAS::graph::zoltan namespace.

Because of the Zoltan behavior, the current process will block until all other involved process also call the distribute() function.
