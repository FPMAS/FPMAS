template<class T>
class FPMAS::graph::DistributedGraph< T >

A DistributedGraph is a special graph instance that can be distributed across available processors using Zoltan.

GhostNode< T > * FPMAS::graph::DistributedGraph< T >::buildGhostNode ( Node< T >  node,
int  node_proc 
)

Builds a GhostNode as an exact copy of the specified node.

For each incoming/outgoing arcs of the input nodes, the corresponding source/target nodes are linked to the built GhostNode with GhostArc s. In other terms, all the arcs linked to the original node are duplicated.

The original node can then be safely removed from the graph.

Parameters
nodenode from which a ghost copy must be created
node_procproc where the real node lives

◆ distribute()

template<class T >
void FPMAS::graph::DistributedGraph< T >::distribute ( )

Distributes the graph accross the available cores using Zoltan.

The graph is distributed using Zoltan::LB_Partition and Zoltan::Migrate. Query functions used are defined in the FPMAS::graph::zoltan namespace.

Because of the Zoltan behavior, the current process will block until all other involved process also call the distribute() function.
