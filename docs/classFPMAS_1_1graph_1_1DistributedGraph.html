template<class T >
int FPMAS::graph::DistributedGraph< T >::distribute ( )

Distributes the graph accross the processors using Zoltan.

Nodes are distributed among processors belonging to the MPI communicator used to instanciate the zoltan instance associated to this distributed graph.

The graph is distributed using Zoltan::LB_Partition and Zoltan::Migrate. Query functions used are defined in the FPMAS::graph::zoltan namespace.

Because of the Zoltan behavior, the current process will block until all other involved process also call the distribute() function.

Returns
Zoltan migrate error code
