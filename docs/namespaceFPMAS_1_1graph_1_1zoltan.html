Detailed Description

The FPMAS::graph::zoltan namespace contains definitions of all the required Zoltan query functions to compute and distribute graph partitions.

◆ node_id()

unsigned long FPMAS::graph::zoltan::node_id ( const ZOLTAN_ID_PTR  global_ids)

Convenient function to rebuild a regular node id, as an unsigned long, from a ZOLTAN_ID_PTR global id array, that actually stores 2 unsigned int for each node unsigned long id. So, with our configuration, we use 2 unsigned int in Zoltan to represent each id. The i input parameter should correspond to the first part index of the node to query in the global_ids array. In consequence, i should actually always be even.

Parameters
global_idsan adress to a pair of Zoltan global ids
Returns
rebuilt node id

ZOLTAN_ID_PTR  global_ids

Writes zoltan global ids to the specified global_ids adress.

The original unsigned long is decomposed into 2 unsigned int to fit the default Zoltan data structure. The written id can then be rebuilt using the node_id(const ZOLTAN_ID_PTR) function.
