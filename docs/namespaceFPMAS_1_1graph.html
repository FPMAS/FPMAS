enum FPMAS::graph::SyncMode

Describes the different graph synchronization modes available.

Once the synchronization mode has been setup, required operations when migrating nodes are automatically performed.

Enumerator
NONE 

In this mode, no overlapping zone is used.

When a node is exported, its connections with nodes that are not exported on the same process are lost.

OLZ 

In this mode, overlapping zones (represented as a "ghost graph") are built and synchronized on each proc.

When a node is exported, ghost arcs and nodes are built to keep consistency across processes, and ghost nodes data is fetched at each GhostGraph::synchronize() call.
