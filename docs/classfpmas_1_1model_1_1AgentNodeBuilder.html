Detailed Description

An api::graph::NodeBuilder implementation that can be used to easily generate a random model, that can be provided to a api::graph::GraphBuilder implementation.

AgentNodes are built automatically, and can be linked according to the api::graph::GraphBuilder algorithm implementation.
